# 🍽️ Dastyor - Restoran Menejer Paneli

> Zamonaviy restoran boshqaruv tizimi - Demo versiya

## 🎯 Loyiha haqida

**Dastyor** - bu O'zbekiston restoranlar uchun ishlab chiqilgan zamonaviy boshqaruv tizimi. Bu **demo versiya** bo'lib, backend siz, faqat frontend qismi bilan ishlaydi.

## ✨ Imkoniyatlar

- 📊 **Dashboard** - Real-time statistika va grafiklar
- 🧾 **Buyurtmalar** - Buyurtmalarni boshqarish
- 🍽️ **Menyu** - Taomlar va kategoriyalar
- 🪑 **Stollar** - QR kod bilan stollar
- 👥 **Xodimlar** - Oshpaz va ofitsiantlar
- ⚙️ **Sozlamalar** - Profil va sozlamalar

## 🚀 Tezkor boshlash

### 1. O'rnatish

```bash
# Klonlash
git clone <repo-url>
cd DastyorMenu

# Paketlar
npm install
```

### 2. Ishga tushirish

```bash
npm run dev
```

Brauzer: `http://localhost:5173`
